How to Use

After building, include these tags in any HTML page:

<link rel="stylesheet" href="https://your-domain.com/cc360-sidebar.css">
<script src="https://your-domain.com/cc360-sidebar.js"></script>

The sidebar will automatically appear on the left side of the page. That's it!

Build Instructions

Run the following command to build the embeddable widget:

npm run build

The built files will be in the dist/ folder. Upload cc360-sidebar.js and cc360-sidebar.css to your hosting service.
